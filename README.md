# Rotate-A-Matrix
Rotate-A-Matrix

Sample Implementation of ratating a matrix.

Problem statement:
1. rotate a matrix clockwise
2. only rotate one step, not 90 degree

1, 2, 3       4, 1, 2
4, 5, 6   =>  7, 5, 3
7, 8, 9       8, 9, 6

1, 2, 3, 4             5, 1, 2, 3
5, 6, 7, 8         =>  9, 10, 6, 4
9, 10, 11, 12          13, 11, 7, 8
13, 14, 15, 16         14, 15, 16, 12
